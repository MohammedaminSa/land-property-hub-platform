# Ethiopian Real Estate Platform

A web-based real estate platform designed for the Ethiopian market, connecting buyers, sellers, landlords, and agents in a secure and trusted environment.

## Project Structure

```
real-estate-project/
├── backend/     # API server and business logic
├── frontend/    # Web application interface
└── docs/        # Project documentation
```

## Features

- **Property Management**: List residential land, apartments, and rental houses
- **User Verification**: Secure registration with document verification
- **Admin Oversight**: Property and user approval system
- **Search & Filter**: Advanced property search capabilities
- **Messaging System**: Direct communication between users
- **Role-based Access**: Buyers, sellers, landlords, agents, and admins

## User Roles

- **Buyer/Tenant**: Browse and inquire about properties of the website
- **Seller/Landlord/Agent**: List and manage properties
- **Admin**: Verify users and approve listings

## Getting Started

1. Navigate to the `backend/` directory to set up the API server
2. Navigate to the `frontend/` directory to set up the web application
3. Check the `docs/` directory for detailed documentation

## Security Features

- JWT authentication
- Password encryption
- Role-based access control
- Document verification system
